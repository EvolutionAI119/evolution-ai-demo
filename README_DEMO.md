## 🔬 核心算法说明

### 1. Bezier 曲面建模

使用 4×4 双三次 Bezier 曲面构建车身三大主要曲面：
- 侧围：v 方向 = 高度（底→顶），u 方向 = 长度（前→后）
- 顶盖：u = 横向，v = 纵向，模拟车顶收缩
- 引擎盖：u = 横向，v = 纵向，含中央凸起

### 2. AI 智能优化

**目标函数**：法向量跳变平方和（光顺度度量）

**算法**：自适应模拟退火
- 扰动：内部控制点的 z 分量（主）+ y 分量（次）
- 接受准则：Metropolis 准则
- 冷却：温度乘以 0.97
- 终止：达到目标光顺度或最大迭代次数

**约束**：
- 保留四个角控制点（边界条件）
- 每次扰动范围衰减（学习率 0.08）

### 3. 质量评估

| 指标 | 计算方法 | A级要求 |
|------|----------|---------|
| **G0** | 位置连续点 | 100% |
| **G1** | 法向量夹角 < 5° | > 85% |
| **G2** | 法向量夹角 < 2° | > 85% |
| **反射线** | 曲率均匀度 + 平滑度 | > 0.7 |
| **综合等级** | G2 占比 + 反射线 | A/B/C/D |

## 🎯 DEMO 限制与未来扩展

- **当前范围**：仅三大主要曲面（侧围/顶盖/引擎盖）
- **未涉及**：完整车身（前后围/车门/底盘/轮拱）、拓扑优化、CAE 仿真
- **优化算法**：当前用模拟退火，可升级为遗传算法 / 贝叶斯优化 / 神经网络代理模型
- **导出格式**：当前为 JSON，可扩展为 IGES/STEP/JT（README 规划格式）
