## 一、整体结构

**核心概念**：不是"展示功能"，而是"让用户感受到化身与他同频共振"

**3幕式结构**：
- **第一幕（0-8s）**：问题导入——"你的数字化身，真的懂你吗？"
- **第二幕（8-65s）**：5个核心场景，节奏递进（战斗→冥想→派对→蒙太奇→休眠）
- **第三幕（65-75s）**：升华收尾——"它不只是工具，它是你的延伸"

**视觉升级要点**：
1. 3D 驾驶舱：用 NURBS 曲面做流线型 UI
2. 镜头运动：每个场景有推/拉/摇/环绕
3. 粒子系统：引擎激活时用粒子光效
4. 场景过渡：用"能量流动"做转场
5. 第一人称视角：加入"用户视角"
