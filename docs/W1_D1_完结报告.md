## 3. 关键技术决策

### 3.1 改用 CDN 引入 element-plus + axios（替代 npm 安装）

**问题**：FUSE overlay writeback 慢，npm install 后 element-plus 488 个文件 0 字节、axios 2-12 个文件 0 字节。

**方案**：在 `index.html` 加 CDN 引用 + 用 alias shim 替换 axios 导入。

**变更**：
- `frontend/index.html`：
  - `<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/element-plus@2.8.1/dist/index.css" />`
  - `<script src="https://cdn.jsdelivr.net/npm/element-plus@2.8.1/dist/index.full.min.js"></script>`
  - `<script src="https://cdn.jsdelivr.net/npm/@element-plus/icons-vue@2.3.1/dist/index.iife.min.js"></script>`
  - `<script src="https://cdn.jsdelivr.net/npm/axios@1.7.4/dist/axios.min.js"></script>`
- `frontend/src/main.ts`：删 `import ElementPlus from 'element-plus'` 改用 `window.ElementPlus`
- `frontend/src/utils/axios-shim.ts`（新文件）：通过 alias 让 `import axios from 'axios'` 走 shim → `window.axios`
- `frontend/vite.config.ts`：
  - `alias: { 'axios': 'src/utils/axios-shim.ts' }`
  - `optimizeDeps.exclude: ['element-plus', 'axios']`

**回切方法**：W1 完工后切回 npm 模式时，删 alias + 删 CDN script + 把 main.ts 改回 `import ElementPlus from 'element-plus'`（4 行 diff）。

### 3.2 修 M2.5 两个老 bug

**Bug 1：get_default_params 不存在**
- 位置：`backend/tasks/optimize_task.py:51`
- 原代码：`from backend.algorithm_compat import get_default_params; parts = build_car(get_default_params())`
- 修复：删除 import，改用 `build_car(None)`（algorithm_model.api.build_car 默认参数就是 CarParams()）

**Bug 2：Trimesh 不是 ndarray**
- 位置：`backend/tasks/optimize_task.py:_make_surface("car_body")`
- 原代码：`arr = np.array(parts[k])` 然后判断 `arr.ndim == 3 and arr.shape[2] == 3`
- 实际：`build_car(None)` 返回 `Dict[str, trimesh.Trimesh]`，不是 ndarray
- 修复：取 `parts[k].vertices`（N, 3 ndarray），判断 `arr.ndim == 2 and arr.shape[1] == 3`

### 3.3 修 W1-D1 E2E 脚本

- presets 期望：4 shapes (sedan/suv/sport/truck) → 3 shapes (sport/luxury/suv)（后端实际）
- /openapi.json：经 vite proxy 失败（vite 不代理 /openapi.json）→ 改直连 8000
- optimize 端点：shape=sedan（凭印象）→ shape=car_body（实际接受的是 sphere/plane_with_noise/cylinder/car_body）
- 202 Accepted：断言 200 → 200|202

## 6. 关键经验沉淀（已更新到 MEMORY/TOOLS）

**本轮新增**：
- **sandbox FUSE overlay 写回慢**：npm install 大包时 488+ 文件被截断为 0 字节
  - 解决：用 CDN 引入 + alias shim（不依赖 npm install 的包）
- **sandbox 4h 回收**：进程 + 库全清
  - 解决：关键服务用 nohup + setsid 启，重启时跑 `apt install redis-server` + `pip install -r requirements.txt`
- **sandbox detached 进程 cwd 重置**：用 `cd` + `nohup` 会丢 cwd
  - 解决：用绝对路径 + 位置参数（不依赖 `cd`）
- **@vue/language-core 2.2.12 自动装上**：避开了 2.1.4 的 yield* bug，不锁版本
- **vue 3.5.x ESM 坑**：node 22 + plugin-vue 触发 `does not provide an export named 'computed'`，patch `vue/index.mjs` + alias
- **M2.5 老 bug**：`get_default_params` 不存在、`Trimesh` 不是 ndarray（M2.5 编码时的两个假设错误）
- **E2E 编写原则**：先查后端 openapi 真实接受参数 + 真实返回，不要凭印象写
