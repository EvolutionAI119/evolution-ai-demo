### 1.3 关键风险（FUSE 0 字节 + sandbox 回收）

**FUSE overlay writeback 慢**是 W1-D1 的真凶：

1. `npm install` 跨 1.5h（1220+ 包）→ 部分文件被截断为 0 字节
2. 0 字节文件集中在 element-plus（488 个）+ axios（2 个）——大包 + 异步写回 = 截断
3. 错误现象不固定：vue-tsc yield* / vue ESM named export / esbuild 94 errors ——**这是文件系统损坏的典型症状**
4. `npm install --force` 不修复（lockfile 命中 cache）
5. `rm -rf` + 重装受 sandbox timeout 影响无法完整执行
6. **sandbox 回收**：18:30-18:36 之间，node_modules 再次被部分清理（element-plus/axios 不存在），Python 库（loguru）和 redis-server 也被回收
