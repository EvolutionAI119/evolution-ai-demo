## 任务目标

创建前端质检页面，可视化展示 NURBS 曲面上的反射光带效果，支持曲率热力图、法向量着色等多种可视化模式。

## 交付物

### 1. 后端 API：`POST /api/v1/quality/reflection-map`
- **文件**：`backend/api/v1/quality.py`（新增路由）
- **请求**：`ReflectionMapRequest`（points + light_direction）
- **响应**：`ReflectionMapResponse`（vertices/normals/curvature/reflection_intensity/indices/score）
- **Models**：`backend/models/quality.py`（新增 ReflectionMapRequest/Response）
- **Service**：`backend/services/quality_service.py`（新增 `reflection_map()` 方法）

### 2. 前端可视化：`docs/reflection_visualizer.html`
- 独立 HTML 文件，Three.js CDN，无需 Python 后端
- 4 种可视化模式：
  - **反射光带**：模拟平行光照射的反射效果
  - **曲率热力图**：蓝→青→绿→黄→红 色彩映射
  - **法向量着色**：RGB 对应 XYZ 分量
  - **线框模式**：显示网格拓扑
- 6 种预设曲面：球面/环面/鞍面/波浪面/圆柱面/车身板
- 光照方向 3 轴滑块实时调节
- 质量指标面板：反射线评分/等级/最大曲率跳变/平均曲率/顶点数
- OrbitControls 支持 3D 旋转/缩放

## 技术细节

### 反射线数据生成
1. 法向量：`estimate_normals()` — 离散网格三角形法向的面积加权平均
2. 曲率：相邻法向夹角的四方向均值
3. 反射强度：|dot(normal, light_direction)|
4. 评分：0.5 × uniformity + 0.5 × smoothness

### 前端渲染
- BufferGeometry：顶点 + 法向量 + 顶点色
- 反射光带效果：sin(dot × 20) 等值线 + 基础光照
- 40×40 网格：1600 顶点，3168 三角面，60fps 流畅
