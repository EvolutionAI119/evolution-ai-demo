## 3. 技术亮点

### 3.1 RMF 双反射法 — 零曲率退化解
传统 Frenet 标架在直线段（κ=0）处退化，法线方向跳变。本方案使用 **Rotation Minimizing Frame（双反射法）**：

1. 初始标架：在首点用任意初始法线 + 叉积构造正交标架
2. 逐点传播：通过两次 Householder 反射将标架从 P_i 平行移动到 P_{i+1}
3. 结果：任意路径（含直线段）都能得到平滑、无扭曲的标架

实测正交性误差 **1.65e-15**（机器精度级别）。

### 3.2 CarBuildRequest 向后兼容设计
新增 freeform 参数不破坏现有 API 调用：

```python
# 旧调用 — 完全兼容
POST /api/v1/car/build {"width": 1.85, ...}

# 新调用 — 带自由变形
POST /api/v1/car/build {"width": 1.85, ..., "freeform": {"preset": "fender_bulge", "amplitude": 0.05}}
```

`CarBuildRequest` 继承原有字段，`freeform: Optional[FreeformParams] = None`，缺失时跳过变形。

### 3.3 性能基准体系
纯 Python 实现，无 Cython/C++ 加速，性能数据代表算法本身复杂度下界：

| 操作 | 耗时 | 目标 |
|------|------|------|
| 基函数求值 | 0.003ms | < 1ms ✅ |
| FFD 5000 顶点 | 18ms | < 50ms ✅ |
| 圆角生成 | 5ms | < 20ms ✅ |
| 扫描曲面 | 15ms | < 30ms ✅ |

若需进一步提升，可引入 NumPy 向量化（曲面求值从 O(n_u × n_v) 循环 → 矩阵运算）或 Cython 编译。
