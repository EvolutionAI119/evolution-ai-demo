## 技术决策记录

### 1. NURBS vs Subdivision Surface
- **选择 NURBS**：汽车行业标配，专利描述也基于 NURBS，控制精度更高
- **不用 Sub-D**：适合角色建模，但汽车行业不常用

### 2. 变形方案：FFD vs 直接控制点编辑
- **选择预设变形**：对 demo 用户更友好，不需要理解 NURBS 控制点
- **保留 API 扩展性**：FreeformDeformation 类支持自定义控制点

### 3. 前端集成方式
- **滑块映射到预设**：每个预设对应 amplitude + center_x + center_z 三个滑块
- **不暴露控制点**：前端用户不需要看到 NURBS 控制网格
