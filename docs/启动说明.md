# 具身智能 3D 动效演示 - 启动指南

## 文件位置
`/app/data/所有对话/主对话/EVOLUTION_AI_DEMO/docs/embodied_3d_demo.html`

## 快速启动

### 方式一：使用本地服务器（推荐）

```bash
# 1. 复制 HTML 文件到图片目录
cp /app/data/所有对话/主对话/EVOLUTION_AI_DEMO/docs/embodied_3d_demo.html \
   /app/data/所有对话/主对话/imgs/260701_00_生图/

# 2. 进入图片目录并启动服务器
cd /app/data/所有对话/主对话/imgs/260701_00_生图/
python -m http.server 8095

# 3. 浏览器打开
http://localhost:8095/embodied_3d_demo.html
```

### 方式二：将图片复制到 HTML 同目录

```bash
# 1. 复制图片到 docs 目录
cp /app/data/所有对话/主对话/imgs/260701_00_生图/img_2.jpg \
   /app/data/所有对话/主对话/EVOLUTION_AI_DEMO/docs/

# 2. 进入 docs 目录启动服务器
cd /app/data/所有对话/主对话/EVOLUTION_AI_DEMO/docs/
python -m http.server 8096

# 3. 浏览器打开
http://localhost:8096/embodied_3d_demo.html
```

## 功能说明

| 功能 | 操作方式 |
|------|---------|
| 视差效果 | 鼠标移动 |
| 旋转视角 | 拖拽鼠标 |
| 缩放 | 滚轮 |
| 自动旋转 | 停止操作 3 秒后恢复 |

## 视觉效果

- **胸口心形核心**：65 个蓝色粒子，呈现心跳般的脉动效果
- **指尖光点**：每手 6 个蓝色光点，微微浮动闪烁
- **能量纹路流光**：35 个粒子沿肩部和腰部纹路流动
- **Bloom 泛光**：蓝色发光部分产生光晕效果
- **2.5D 起伏**：图片表面有轻微的 3D 起伏动画

## 技术栈

- Three.js r167
- Post-processing (EffectComposer + UnrealBloomPass)
- Custom ShaderMaterial for particles
- OrbitControls for camera interaction
