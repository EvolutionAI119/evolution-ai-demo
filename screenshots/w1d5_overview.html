✅ CDN 模式（保留）

<!-- index.html 4 行 CDN 引用 -->
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/element-plus@2.8.1/dist/index.css" />
<script src="https://cdn.jsdelivr.net/npm/element-plus@2.8.1/dist/index.full.min.js"></script>
<script src="https://cdn.jsdelivr.net/npm/@element-plus/icons-vue@2.3.1/dist/index.iife.min.js"></script>
<script src="https://cdn.jsdelivr.net/npm/axios@1.7.4/dist/axios.min.js"></script>
<script src="https://cdn.jsdelivr.net/npm/vue-i18n@9.14.5/dist/vue-i18n.global.prod.js"></script>

// main.ts: window.ElementPlus / window.ElementPlusIconsVue
app.use(window.ElementPlus, { locale: zhCn })
// → Vite ready 2.7s · 30/30 E2E 过 ✅
💡 决策：保留 CDN（element-plus 0 字节 sandbox FUSE 老问题）。下次切回时机：M4 Docker 化在镜像内 npm install 不受 FUSE 影响。i18n 历史 bug（孤立 `} as const`）已顺手修，下次切回 npm 直接用。
